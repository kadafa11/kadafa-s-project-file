Project Overview

This project showcases data analysis skills using Microsoft Excel, applied to three datasets from Kaggle: Wall-Mart Sales, Stock Dataset, and Diabetes Dataset. The tasks performed span data cleaning, advanced Excel functionalities, data modeling, and visualization. Below is a breakdown of the steps undertaken and their outputs.

Datasets Used

1. Wall-Mart Sales: Sales data including categories, regions, and revenue.


2. Stock Dataset: Data on stock performance over time.


3. Diabetes Dataset: Medical dataset with attributes related to diabetes diagnostics.

Tasks Performed

a. Data Cleaning

Removed duplicates and irrelevant data.

Handled missing values by using appropriate imputation techniques.

Standardized inconsistent formats (e.g., date formats, text casing).


b. Conditional Formatting

Applied color scales to visualize trends (e.g., sales performance and stock changes).

Highlighted outliers or specific conditions (e.g., diabetes risk levels).


c. Advanced Sort and Filter

Utilized Advanced Sort to arrange data by multiple criteria (e.g., sales by region and month).

Used Advanced Filter to extract specific subsets of data (e.g., high-revenue stores or stocks exceeding a certain threshold).


d. Text Functions

Applied Excel functions to analyze data efficiently:

SUMIF/SUMIFS: Summed values based on single or multiple conditions.

COUNTIF/COUNTIFS: Counted occurrences of specific conditions.

AVERAGEIF/AVERAGEIFS: Calculated averages under given criteria.


e. Data Modeling Using Power Pivot

Combined datasets in Power Pivot to create relationships between tables.

Enhanced data analysis capabilities through calculated fields and measures.


f. Pivot Tables, Charts, and Dashboard

Created pivot tables and charts to summarize key metrics.

Built an interactive dashboard featuring six visuals, including:

1. Sales trends over time.


2. Regional stock performance comparison.


3. Average sales by category.


4. Diabetes risk distribution.


5. Top-performing stores.


6. Stock price volatility.


Insights:
Key findings were derived from visual trends and comparative analysis.

g. Box and Whiskers Plot

Visualized a dataset using a Box and Whiskers Plot to identify distribution and outliers.


h. Correlation and Regression Analysis

Analyzed relationships between variables (e.g., sales and advertising spend, glucose levels and BMI).

Built regression models to predict outcomes and determine key influencing factors.

Tools Used

Microsoft Excel: Primary tool for data cleaning, analysis, and visualization.

Power Pivot: For advanced data modeling and relationship building.

Kaggle: Source of datasets.

Project Highlights

1. Demonstrated data cleaning and preparation techniques.


2. Showcased proficiency with Excel's advanced formulas and conditional formatting.


3. Built an interactive dashboard with actionable insights.


4. Conducted statistical analyses (Correlation and Regression) to derive patterns and trends.
