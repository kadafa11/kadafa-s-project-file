# CORRELATION: A correlation matrix is a table that shows the correlation coefficients between two variables.
# The coefficient correlation ranges from -1 to 1:
# A:(1) Perfect positive correlation(as one variable increases, the other also increases in a perfectly linear manner)
# B:(0) No correlation (the variables do not have a linear relationship)
# C:(-1) Perfect negative correlation( as one variable increases, the other decreases in a perfectly linear manner)
# BLUE: Indicates negative correlations (closer to -1)
# RED: Indicates positive correlations (closer to 1)
# White or Light Colors: indicates weak or no correlation (closer to 0)
# DIAGONAL LINE: The diagonal cell from top-left to the bottom-right will always have a value of 1 (Perfect positive correlation) because each variable
# is perfectly correlated with itself.

# ANNOTATIONS; The annot = True parameter adds the exact correlation coefficients to each cell in the HEATMAP, 
# making it easier to read the Numerical values.
In [19]:
# INTERPRETATION OF HEATMAP

# Strong Positive Correlation (Red):
# If two variables have a correlation close to 1 (dark red), it means that as one variable increases, the other variable also increases in a predictable manner

# Strong Negative Correlation (Blue):
# If two variables have a correlation close to -1 (dark blue), it means that as one variable increases,the other variable decreases in a predictable manner.

# Weak or No Correlation (White or Light Colors), It means that there is no linear relationship between the two variables.

          # e.g Age and Salary: 
# A dark red cell at the intersection of Age and Salary mean older individuals tend to have higher salaries.
# A dark blue cell at the intersection of Age and Salary means younger individuals tend to have lower salaries.
# A white cell means there's no linear relationship between Age and Salary.
